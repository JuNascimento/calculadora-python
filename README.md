# calculadora-python


## Bem vinda(a) a minha Calculadora em Python!

### Tecnologias usadas:

- [x] Python 3.8.3
- [x] Pipenv
- [x] Pyenv


### Como configurar o projeto

````
make setup
````


### Como rodar o projeto

````
make run
````


### Como validar o lint

````
make lint
````

**O lint é executado toda vez que o projeto é executado**
